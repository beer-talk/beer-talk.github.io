### Yêu cầu hệ thống

Xin chào tất cả mọi người, để đảm bảo Workshop diễn ra trơn tru, hãy chuẩn bị các công cụ sau:
1. **Node.js v20+** hoặc Python 3.11+
2. **Docker Desktop** (cho việc dựng vector database cục bộ)
3. API key (OpenAI hoặc Anthropic) nếu bạn đã có, nếu chưa BTC sẽ cung cấp key tạm thời.

Chúng ta sẽ trải qua 60 phút code liên tục để build một support bot có khả năng truy xuất file (RAG) và tự động gọi action (function calling).

Gặp lại mọi người!
